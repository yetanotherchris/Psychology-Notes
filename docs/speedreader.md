# Speedreader

This tool provides speed reading functionality with adjustable reading speeds and text formatting.

**[Launch Speedreader Application](../tools/speedreader/){:target="_blank"}**

## Features

- Adjustable reading speed (words per minute)
- Text highlighting and focus modes
- Customizable display settings
- Progress tracking

The application runs independently from this documentation site and provides a focused environment for practicing speed reading techniques.
