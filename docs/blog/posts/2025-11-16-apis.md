# APIs to search acadamic papers

There are quite a few alternatives to Google Scholar that are free. This is how the [The academic search](https://psychology-notes.fly.dev/tools/academic-search/) on this site was built.

<!-- more -->

- [CrossRef.org API](https://api.crossref.org/swagger-ui/index.html). This searches academic papers and authors.
- [Semantic Scholar search](https://www.semanticscholar.org/search?q=%22D.%20Winnicott%22%20mirror&sort=relevance)
- [Citations API](https://citation.doi.org/api-docs.html)
